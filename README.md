# Analyse-de-tweet-US-election-2020-
Ce projet vise à analyser des tweets liés à l’élection présidentielle américaine de 2020 (Trump vs Biden). Il comprend le nettoyage des données, une analyse de sentiments à l’aide de modèles Hugging Face et une étude des utilisateurs afin d’identifier les tendances d’opinion et l’influence sur Twitter

## Statut Actuel du Projet
